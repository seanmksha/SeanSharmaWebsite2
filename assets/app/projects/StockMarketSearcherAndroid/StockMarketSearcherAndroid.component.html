The android application communicates with a backend webserver (Java to PHP). The application does all the front-end GUI displays. All of the information is gathered from 3rd party APIs via a PHP page hosted on google cloud and sent through JSON Objects.

Some features included:

  • Saving favorites and loading up the stock statistics for each favorite on startup
  • Charts on past stock history
  • News feed related to the searched stock (available by Bing)
  • Autocomplete which searches through stock names
  • Autorefresh which updates the data every few seconds

Krys Towers Web Development



Krys Towers Web Development was made from scratch by me to perform freelance web services for clients. Typically clients required a website for their small business, and CMS features to post updates to their customers.



Most of the Krys Towers Web Development website are static html pages. The animations on the main page are done through a mixture of javascript and css. Initially, the website started out as a simple 5 page website, then grew into a full-blown detailed website. The payment part of the website found at pay.krystowers.com saves existing customers, allows customers to update their information, and securely sends customers to my Bank's merchant service payment gateway for payments.
